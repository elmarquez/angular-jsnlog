Angular JSNLog

Capture client side log messages and forward them to your server for storage and processing. angular-jsnlog is a drop-in replacement for Angular's $log service, provided by the amazing JSNLog library.

Use Bower to automatically install angular-jsnlog and its dependencies in your project.

bower install -S angular-jsnlog

Refer to the README.md document for instructions on how to install and configure logging.

Try It Out

Click a button to create a new logging event. JSNLog will batch and then send the log message to the remote server. The log view polls the remote server every few seconds and shows you the updated log.
