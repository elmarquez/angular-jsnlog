Angular JSNLog

JSNLog is a fantastic client side logging library. It enables you to capture all log output from your and forward those log messages to a server for storage and processing. angular-jsnlog is a drop-in replacement for Angularjs's $log service.

Use Bower to automatically install the library and its dependencies in your project.

bower install -S angular-jsnlog

Refer to the README.md document for instructions on how to create and customize logging. Examples are available from the repository.
